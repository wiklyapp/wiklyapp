Βρείτε Χρόνο: Έξυπνο πρόγραμμα

Τι είναι το "Βρείτε Ελεύθερο Χρόνο";

Το "Βρείτε Ελεύθερο Χρόνο" σας βοηθά να εντοπίσετε διαθέσιμες ώρες στο πρόγραμμά σας για να προγραμματίσετε νέες εργασίες, οικογενειακές συναντήσεις, κοινές δραστηριότητες ή άλλα γεγονότα που απαιτούν συντονισμό.

Πώς λειτουργεί:

  1. Επιλέξτε συμμετέχοντες

    Διαλέξτε μέλη της οικογένειας ή άτομα που πρέπει να είναι διαθέσιμα για το προγραμματισμένο γεγονός.

  2. Επιλέξτε ώρα

    Επιλέξτε "Πρωί" ή "Απόγευμα" για να περιορίσετε την αναζήτηση.

  3. Δείτε τη διαθεσιμότητα
    • Σημειωμένες ώρες: Υποδεικνύουν τις ώρες που είναι διαθέσιμες για όλους τους συμμετέχοντες.
    • Κενές ώρες: Υποδεικνύουν ώρες όπου τουλάχιστον ένας συμμετέχων δεν είναι διαθέσιμος.
    • Γκρι ώρες: Υποδεικνύουν ώρες που έχουν ήδη περάσει.

  4. Επιλέξτε ημέρα και ώρα

    Κάντε κλικ σε μία σημειωμένη ώρα που εξυπηρετεί όλους.

  5. Δημιουργία εργασίας

    Μόλις επιλέξετε την ώρα, ακολουθήστε τις οδηγίες για να προσθέσετε εύκολα μία νέα εργασία στη συγκεκριμένη ώρα.

Συμβουλές

  • Για καλύτερα αποτελέσματα, βεβαιωθείτε ότι όλοι οι συμμετέχοντες έχουν ενημερώσει τα προγράμματά τους στην εφαρμογή. Το "Βρείτε Ελεύθερο Χρόνο" λαμβάνει υπόψη υπάρχουσες εργασίες και ραντεβού για να καθορίσει τη διαθεσιμότητα.
  • Μετακινηθείτε στις ημερομηνίες για να βρείτε διαθέσιμες ώρες τις επόμενες εβδομάδες ή μήνες.
  • Ρυθμίστε την προτίμηση ώρας (πρωί/απόγευμα) ή αλλάξτε τους συμμετέχοντες για πιο ακριβή αναζήτηση.

Απολαύστε τη λειτουργία "Βρείτε Ελεύθερο Χρόνο" και οργανώστε τις οικογενειακές σας εκδηλώσεις με ευκολία!
